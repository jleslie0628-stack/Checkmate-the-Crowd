Checkmate the Crowd ♟️👥

Checkmate the Crowd is a real-time, interactive chess experience where one player faces off against a "Hive Mind" of voters. While "The One" makes individual moves, "The 100" (the crowd) must coordinate and vote on every move to decide their collective strategy.

🚀 How It Works
The One: You play as White, executing moves traditionally on a standard chessboard.

The 100: You play as Black, voting on moves alongside other players. The move with the highest consensus is executed automatically.

Real-time Collaboration: Powered by Socket.io, moves, chat, and timers update instantly for everyone in the room.

🛠️ Features
Dual-Role Gameplay: Switch between the solo strategist and the collective intelligence.

Voting System: Crowd-sourced decision-making for the Black pieces.

Hive Mind Chat: Coordinate tactics with your fellow crowd members.

Responsive Design: Fully optimized for both desktop and mobile play.

Global Timer: Keeps the pressure on with synchronized turn limits.

💻 Tech Stack
Frontend: HTML5, CSS3, JavaScript, chessboard.js

Backend: Node.js, Express

Real-time Engine: Socket.io

📋 Installation
Clone the repository:

Bash
git clone https://github.com/jleslie0628-stack/Checkmate-the-Crowd
Install dependencies:

Bash
npm install

Start the server:

Bash
node server.js
Open your browser and navigate to http://localhost:3000.
